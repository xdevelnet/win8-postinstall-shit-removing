Этап установки

1) Запишите ключ активации заранее
2) Отключите компьютер от интернета путем извлекания сетевого кабеля
3) Загрузитесь с загрузочного носителя для установки Windows
4) Выбирайте установочные опции как обычно вы это делаете при установке Windows
5) После диалога выбора цвета и имени компьютера установщик предложит выбрать параметры. Не стоит соглашаться на стандартные: нажмите на кнопку "настроить".

	Поле со списком "Центр обновления Windows" следует установить в "Не настраивать Центр обновления Windows".
	Все переключатели ставим в отключено кроме "Отправлять запрос Do Not Track".
	Кнопка "Далее".
	Тут по желанию. Лично я убрал "Использовать отчеты об ошибках Windows для поиска решений проблем в Интернете". Остальные пункты уже стояли на "Отключено".
	Кнопка "Далее".
	Отключаем всё.
	Кнопка "Далее".
	Заполняем поля на ваше усмотрение.

6) Через некоторое время попадаем в интерфейс Metro. Тащим мышь в правый верхний угол, ждем появления боковой менюшки. Выбираем "Параметры" -> "Изменение параметров компьютера".
7) "Компьютер и устройства" -> "Автозапуск" -> "Использовать автозапуск для всех носителей и устройств" -> "Откл."
8) "Поиск и приложения" -> "Поделиться" (кстати, я хз что имеется ввиду) -> "Делиться через эти приложения" -> Выключаем всё
9) "Конфиденциальность" -> "Веб-камера" -> "Разрешить приложениям использовать мою веб-камеру" -> "Откл."
10) "Конфиденциальность" -> "Микрофон" -> "Разрешить приложениям использовать мой микрофон" -> "Откл."
#11) "Обвновление и восстановление" -> "Центр обновления Windows" -> "Выберите тип установки обновлений" -> "Важные обновления" -> "Не проверять наличие обновлений"
11) "Проводник" -> Правой кнопкой мыши по "Этот комьютер" -> "Свойства" -> Получаем окно "Система"
12) В окне "Система" выбираем "Настройка удаленного доступа" -> "Удаленный доступ" -> "Разрешить подключения удаленного помощника к этому компьютеру" -> Снимаем галочку
13) (по желанию) В окне "Система" выбираем "Панель управления - домашняя страница" -> "Просмотр: Мелкие значки" -> "Брандмауэр Windows" -> "Включение и отключение Брандмауэра Windows" -> "Отключить брандмауэр Windows" для обоих типов сетей
14) ОС имеет отдельный "режим повышенной безопасности", в котором брандмауэр всё равно работает. Его можно отключить (если хотите) в пункте "Дополнительные параметры" -> "Свойства брандмауэра Windows" -> "Состояние" -> "Брандмауэр" -> "Отключить"
15) В окне "Система" выбираем "Администрирование" -> "Службы" -> Ищем службу "Windows search" -> "Свойства" -> Меняем тип запуска на "Отключена" -> Жмем кнопку "Остановить"
16) В проводнике открываем свойства каждого из дисков -> Вкладка "Общие" -> Снимаем галочку с "Разрешить индексировать содержимое файлов на этом диске в дополнение к свойствам файла" -> Соглашаемся в новом диалоговом окне, пропускаем недоступные файлы.
17) Аналогично с пунктом выше в окне свойств диска выбираем вкладку "Сервис" -> Кнопка "Оптимизировать" -> В новом окне нажимаем на кнопку "Изменить параметры" -> Снимаем все галочки -> Кнопка "OK". Если через большой период использования системой вы заметите снижение дисковой производительности из-за большой фрагментации (а она возможна когда вы пожлобились на достаточный объем места для системного раздела) - можете провести дефрагментацию из этого окна вручную в удобное для вас время.
18) Защитник Windows тоже выбрасываем на помойку. Какого чёрта система должна следить за программами? Она должна выполнять их!
19) Удаляем все имеющиеся приложения Windows
20) Выключаем магазин: http://advanceduser.ru/obuchenie-windows-8/kak-otklyuchit-magazin-windows.html
20a) reg add "HKEY_CURRENT_USER\Software\Policies\Microsoft\WindowsStore" /v RemoveWindowsStore /t reg_dword /d 1 && gpupdate /force
Команда взята отсюда: http://www.cyberforum.ru/windows8/thread1394020.html
21) Устанавливаем classic shell и браузер вместо IE (firefox либо ungoogled-chromium)
22) Выбрасываем на помойку IE через "Компоненты windows"
23) Выключаем гибернацию при действии "завершение работы" - вместо этого будет нормальное выключение: Панель управления - Электропитание - Действие кнопок питания - Изменение параметров, которые сейчас недоступны - Включить быстрый запуск - Снимаем галочку
